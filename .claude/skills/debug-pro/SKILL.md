# Debug Pro

## Phase 1: Reproduce
- Exact error message + stack trace
- Steps to reproduce
- Environment details

## Phase 2: Isolate
- Binary search codebase — narrow to one function
- Check: data? logic? async timing? type mismatch?

## Phase 3: Fix
- Explain WHY the bug happened
- Write regression test FIRST
- Apply minimal fix
